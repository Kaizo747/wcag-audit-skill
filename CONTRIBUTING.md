# Contributing

Thanks for your interest in improving this skill.

## How to contribute

- Open an issue for bugs, missing checks, or unclear instructions.
- Suggest improvements to the skill workflow or report format.
- Submit a pull request with a clear description of the change.

## Before submitting

- Keep the language simple and practical.
- Do not remove required report sections unless there is a clear improvement.
- Make sure examples still match the skill instructions.

## Good contributions

- Better report wording.
- More examples.
- Improved manual checklist guidance.
- Better GitHub documentation.
